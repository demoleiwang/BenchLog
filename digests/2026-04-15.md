## Trending benchmark-related papers (not yet promoted)

1. **[LARY: A Latent Action Representation Yielding Benchmark for Generalizable Vision-to-Action Alignment](https://arxiv.org/abs/2604.11689)** — Nie et al., submitted 2026-04-13.
   Proposes a *new* vision-to-action benchmark with > 1M videos (1,000 hours), 620K image pairs, 595K motion trajectories across 151 action categories. Key finding: general-purpose vision models without action supervision outperform specialized embodied models. **Promote? Maybe — re-evaluate next run based on community pickup. Category would be new: `embodied` or `robotics`.**

2. **[ClawGUI: A Unified Framework for Training, Evaluating, and Deploying GUI Agents](https://arxiv.org/abs/2604.11784)** — Tang et al., submitted 2026-04-13. 307 upvotes on HF.
   Framework + eval suite for GUI agents; ClawGUI-2B reportedly achieves 17.1% on "MobileWorld GUI-Only". Positioning is as a framework using existing benchmarks rather than a new benchmark itself. **Promote? No — fits digest-only, but cross-reference when writing a future MobileWorld entry.**

3. **[Habitat-GS: A High-Fidelity Navigation Simulator with Dynamic Gaussian Splatting](https://arxiv.org/abs/2604.12626)** — Xia et al., submitted 2026-04-14.
   Extends Habitat-Sim with 3D Gaussian Splatting for photorealistic rendering. Simulator / environment, not a new benchmark per se. **Promote? No — note in the Habitat benchmark family when first added.**
